### Info 🐢

Repo for [honuli.com](https://honuli.com)

### Maintenance 🔨

Run locally:
```bash 
hugo server
````

Fix layer issues:
```bash
git submodule init
git submodule update
```

Update project to latest [PaperMod](https://github.com/adityatelange/hugo-PaperMod) version:
```bash
git submodule update --remote --merge
```
